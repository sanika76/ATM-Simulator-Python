# 🏧 ATM Simulator System (Python | OOP | File Handling)

## 📌 Project Overview
The ATM Simulator is a Python-based application developed using Object-Oriented Programming (OOP) and File Handling concepts.  
This project simulates real-world ATM functionalities such as secure PIN authentication, balance management, deposit, withdrawal, and mini statement generation with persistent data storage.

---

## 🎯 Key Features
- 🔐 Secure 3-Attempt PIN Authentication System  
- 💰 Balance Check Functionality  
- 💵 Deposit Money Feature  
- 💸 Withdraw Money with Insufficient Balance Handling  
- 📜 Mini Statement (Last 5 Transactions Tracking)  
- 💾 File Handling for Permanent Data Storage  
- 📋 Menu-Driven User Interface (CLI Based)  

---

## 🛠️ Technologies Used
- Python  
- Object-Oriented Programming (OOP)  
- File Handling (Text Files)  
- os Module  
- Command Line Interface (CLI)

## 📊 Project Functionalities (Numerical - ATS Optimized)

## 💡 How It Works
- User enters name and sets ATM PIN
- System verifies PIN (Max 3 attempts)
- User can choose options from menu:
- Check Balance
- Deposit Money
- Withdraw Money
- Mini Statement
- Change PIN
- All data (Balance, PIN, Transactions) is saved automatically using file handling

---

## 🔒 Security Features
- Private variables for PIN and Balance (__pin, __balance)
- 3-attempt card blocking simulation
- Input validation for invalid amounts
- Error handling for insufficient balance
