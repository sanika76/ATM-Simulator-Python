# 🏧 ATM Simulator (Python Project)

## 📌 Project Overview
This is a Python-based ATM Simulator built using Object-Oriented Programming and File Handling. It simulates real ATM functionalities like deposit, withdrawal, balance check, mini statement, and PIN authentication.

## 🚀 Features
- Secure 3-attempt PIN authentication
- Deposit and Withdraw functionality
- Balance Check System
- Mini Statement (Last 5 Transactions)
- File Handling for data storage
- Menu-driven interface

## 🛠️ Technologies Used
- Python
- OOP (Object-Oriented Programming)
- File Handling
- os Module

## 🎯 Project Purpose
This project was created to practice Python, OOP concepts, and real-world system design for learning and resume building.
